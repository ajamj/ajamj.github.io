## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Backend / server-side processing | Static site only |
| User authentication | Not needed for portfolio |
| CMS / blog with dynamic content | Simple static pages preferred |
| Multi-language support | English only for v1 |
| E-commerce / paid features | N/A |
| Complex animations | Keep it fast and professional |
