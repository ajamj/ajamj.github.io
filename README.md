# Ajam Jamaludin | Geophysics Researcher Portfolio

Computational seismology, earthquake early warning systems, and ML-based seismic parameter estimation.

## 🚀 Overview
This is a high-performance, accessible, and editorial-style portfolio built with **Astro 5**. It features a modern **Obsidian-Emerald** dark theme and fluid responsive layouts designed for the academic and research community.

## 🛠 Tech Stack
- **Framework**: [Astro](https://astro.build/)
- **Styles**: Vanilla CSS with [Tailwind CSS](https://tailwindcss.com/) (v4)
- **Data Viz**: [D3.js](https://d3js.org/)
- **Hosting**: [GitHub Pages](https://pages.github.com/)
- **Optimization**: Astro Image Service (WebP/Avif)

## 📁 Structure
- `src/pages/`: Main site pages (Home, About, Projects, Blog).
- `src/components/`: Reusable Astro components (Hero, Timeline, Grids).
- `src/content/`: Blog posts and configurations using Astro Content Collections.
- `src/styles/`: Global CSS and design tokens.

## 🧪 Development
```bash
# Install dependencies
npm install

# Start local dev server
npm run dev

# Build for production
npm run build
```
